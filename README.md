Instituto Politécnico da Huíla - UMN

2º Ano | Engenharia Informática.

## Descrição

Este projeto implementa diferentes algoritmos de ordenação aplicados a um sistema de Quadro de Honra dos Estudantes, utilizando a linguagem Python.

Os estudantes são classificados de acordo com as suas notas em ordem decrescente.  
Após a ordenação, é apresentada a classificação geral dos estudantes.  
Os estudantes com nota igual ou superior a 15 valores recebem Menção Honrosa.

## Algoritmos Implementados

- Bubble Sort
- Selection Sort
- Insertion Sort
- Shell Sort
- Merge Sort
- Quick Sort

## Critério de Classificação

- Todos os estudantes são ordenados por nota.
- A classificação é exibida do maior para o menor valor.
- Estudantes com nota igual ou superior a 15 valores recebem Menção Honrosa.
